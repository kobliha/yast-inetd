Features

The inetd module handles configuration for inetd and xinetd.

Inetd

The configuration of inetd is stored in /etc/inetd.conf. All services are specified in this file.

XInetd

The configuration of xinetd is stored in /etc/xinetd.conf and in /etd/xinetd.d directory. Each service has configuration stored in his own file there in this directory.
Internet super-server (inetd and xinetd) is executing services (like finger, echo, telnet...) acording to the network requests.
